Bold & Strong

The Bold and Strong elements do essentially the same thing, make the text bold. The key difference is that the Strong element will be read out in a stronger tone when using a screen-reader. For this reason, Strong is typically preferred in programming.

Bold is represented by <b>

Strong is represented by <strong>

example using <b> and <strong>:

This line is written using <b>

This line is written using <strong>

Italic & Emphasis

Similar to Bold and Strong, Italic and Emphasis are essentially the same however Emphasis will be read with more emphasis by the screen-reader.

Italic is represented by <i>

Emphasis is represented by <em>

example using <i> and <em>:

This line is written using <i>

This line is written using <em>

The Mark element will highlight the text which it surrounds.

Small

The small element will change the text size to be smaller than the default text size.

example using <small>:

This text is smaller than usual.

Deleted Text

The name of this element is a little misleading as it won't actually delete the contents within, rather it will strike it out.

Deleted text is represented by <del>

example using <del>:

This text has been stricken out using <del>.

Inserted Text

Inserted text will add an underline to the text it surrounds

Inserted Text is represented by <ins>

example using <ins>:

This text is underlined.

Subscript & Superscript

Subscript and Superscript are useful when writing algebra and scientific terms to make numbers and letters appear above and below their regular place.

Subscript is represented by <sub>

Superscript is represented by <sup>

example using <sub> & <sup>:

Subscript: H2O

Superscript: 52

Keyboard

Keyboard is represented by <kbd>
